  4. Pass a function as an argument

    Practice a higher-order function by passing filtering rules into one reusable function.

    • Create a function named printMatchingScores.
    • It must accept a list of scores, a label, and a predicate function: (Int) -> Boolean.
    • Inside the function, use filter with the predicate and print the matching scores.
    • Call the function three times: passing scores, failing scores, and perfect scores.

    Starter idea

    fun printMatchingScores(
    scores: List<Int>,
    label: String,
    rule: (Int) -> Boolean
) {
    val matches = scores.filter(rule)
    println("$label: $matches")
}

  5. Mini project: study task analyzer

    Combine val/var, functions, filters, and higher-order functions in one program.

    • Create a data class StudyTask(val title: String, val minutes: Int, val completed: Boolean).
    • Create at least six study tasks.
    • Use var to track total completed minutes as you process completed tasks.
    • Write a higher-order function named tasksMatching that accepts a list of tasks and a predicate: (StudyTask) -> Boolean.
    • Use tasksMatching to print completed tasks, incomplete tasks, and tasks that take 30 minutes or more.
    • Print the total minutes spent on completed tasks.

    Starter idea

    data class StudyTask(
    val title: String,
    val minutes: Int,
    val completed: Boolean
)

fun tasksMatching(
    tasks: List<StudyTask>,
    rule: (StudyTask) -> Boolean
): List<StudyTask> {
    return tasks.filter(rule)
}

fun problem5() {
    val tasks = listOf(
        StudyTask("Read Kotlin notes", 25, true),
        StudyTask("Practice filters", 35, false),
        // Add at least four more tasks.
    )

    var completedMinutes = 0
    val completed = tasksMatching(tasks) { it.completed }

    completed.forEach { task ->
        completedMinutes += task.minutes
    }

    println("Completed tasks: $completed")
    println("Completed minutes: $completedMinutes")
}
